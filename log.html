LoggingÂ¶

The Addon class provides 4 methods for simplified logging: log_notice, log_warning, log_error and log_debug. Those methods write messages to the Kodi log with the respective levels. The messages include the addon ID to help distinguish messages belonging to the current addon. All methods accept both str and unicode strings.
